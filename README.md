# 🛒 RetailGuard AI – Product Analysis

## 📌 Project Overview

RetailGuard AI is a Data Analyst portfolio project that analyzes a retail product dataset containing 10,000 records. The project demonstrates SQL analysis using PostgreSQL and interactive dashboard creation using Power BI.

---

## 🎯 Objective

- Analyze retail product data.
- Solve business problems using SQL.
- Build an interactive Power BI dashboard.
- Present business insights for decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Excel
- PostgreSQL (pgAdmin)
- SQL
- Power BI

---

## 📂 Dataset Information

- Records: **10,000**
- Columns:
  - Product_ID
  - Product_Name
  - Category
  - Brand
  - Cost_Price
  - Selling_Price
  - Supplier_ID
  - Launch_Date
  - Status

---

## 📊 Business Questions Solved

- Display all products.
- Display active products.
- Products with Selling Price ≥ ₹10,000.
- Products from Logitech, Sony, and Samsung.
- Total number of products.
- Average selling price by category.
- Total selling price by category.
- Product count by category.
- Minimum and maximum selling price by category.
- Top 10 most expensive products.
- Top 10 cheapest products.
- Unique brands and categories.
- Categories with average selling price greater than ₹5,000.

---

## 📈 Power BI Dashboard

The dashboard includes:

- Total Products KPI
